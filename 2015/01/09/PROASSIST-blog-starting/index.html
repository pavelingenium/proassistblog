Introducing PROASSIST

Information about new features will be presented here.

What you can do with PROASSIST

Create assistance cases of various types
Find workers using smart rules
Take advantage of geo-routing capabilities
Speed up routine tasks, avoid mistakes
Be in touch with your end consumer via notifications
Use configurable job templates
Automate data validations, e.g. in database of your clients
Export invoicing data to your billing system
Have filed history of all work you have done

The daily routine of call-center

Accept incoming requests from clients
Create new cases, validate client’s policy numbers
Create specific jobs in case using configurable templates
Find nearest worker for job, validate routes, compute distances
Communicate with worker via SMS, email, mobile app
Client first: control the case processing, check for late arrivals, track progress on map
Compute the price of the job, distinguish between covered price and total price
Import photos, generate documents
Export invoicing data to accounting software

What tools and languages we use to create PROASSIST

Microsoft .NET business components written in C#
Microsoft SQL Server or Windows Azure database storage
NodeJs website with Jade templating
AngularJs front end
Leaflet for iteractive maps
Open street map data for routing
MapBox for displaying the routes and assets on map
